# My First Algorithm 🚀

This is the very first code I ever wrote! I'm starting my journey into the world of programming and logic.

## 📝 Description
This project contains a basic algorithm developed in **VisualG**. It's a simple exercise to practice:
- Variable declaration
- User input/output (`leia` and `escreva`)
- Arithmetic operations (Sum, Average)
- Math functions (Power and Square Root for calculating a Triangle's Hypotenuse)

## 🛠️ Concepts Learned
- **Sequential Logic:** Understanding that the computer follows a step-by-step process.
- **Data Types:** Using Integers, Reals, and Characters.
- **Math Logic:** Implementing formulas like the Pythagorean theorem.

## 💻 How to run
1. Download and install [VisualG](http://visualg3.com.br/).
2. Open the `.alg` file.
3. Press `F9` to run.
